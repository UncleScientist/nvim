# Toggle nvim config

This is evolving all the time so who knows

## To install

- `Clone into ~/.config`
- `cd ~/.config/nvim`
- run `setup.sh --init`

To use FZF, install the [FZF](https://github.com/junegunn/fzf) package.

## To update

- `cd ~/.config/nvim`
- run `setup.sh` without the --init argument
